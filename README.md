# Hey Amigo - AI Voice Notebook & IDE

An AI-powered learning assistant that helps users:

- Convert speech to text
- Correct notes automatically
- Explain concepts interactively
- Chat with AI study assistant
- Create and manage smart timetables
- Generate AI-based study plans
- Track learning with dashboard

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript

## Features
- 🎤 Speech to Text
- ✍️ Auto Note Correction
- 🧠 Concept Explainer
- 🤖 AI Chat Assistant
- 📅 Smart Timetable
- ⚡ Auto Timetable Generator
- 📊 Weekly Dashboard

## Note
Sensitive files like database and uploads are excluded for security.
